# Car-Design
***An AI running car***
This project is about creating an ***AI-powered driver safety system*** that helps reduce road accidents. Many accidents happen because drivers get tired, distracted, or fail to notice something dangerous on the road. The goal of this project is to build a system that can monitor both the driver and the road environment and warn the driver before a dangerous situation occurs.
The video from these cameras is processed by a small computer, such as a ***Raspberry Pi*** or ***similar device***. Using computer vision techniques, the system analyses the images in real time. It can recognise objects on the road and also detect important facial features of the driver, such as ***eyes***, ***head position***, and ***movement***.
After analysing the information, the system calculates a risk level for the current driving situation. For example, if the driver looks sleepy while the car is moving fast or if a pedestrian suddenly appears in front of the vehicle, the system identifies this as a ***high-risk situation***.
